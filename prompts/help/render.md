# render — render a diagram element to SVG

## SYNOPSIS
    syscribe -m <root> render <diagram-path>

## DESCRIPTION
Renders a Diagram element's server-side SVG to stdout. For the multi-step
companion-SVG authoring workflow, see `diagram`.

## EXAMPLES
    syscribe -m model/ render model/Diagrams/SystemBDD.md

    diagram
